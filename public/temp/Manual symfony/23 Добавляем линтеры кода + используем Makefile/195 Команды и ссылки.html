Установить пакет friendsofphp/php-cs-fixer:

symfony composer require friendsofphp/php-cs-fixer --dev


Установить пакет phpstan/phpstan:

symfony composer require phpstan/phpstan --dev


Установить пакет eslint для работы с vue.js:

npm install --save-dev eslint eslint-config-prettier eslint-plugin-prettier eslint-plugin-vue


Запуск сценария "check" с игнорированием статуса ошибки:

make check --keep-going
