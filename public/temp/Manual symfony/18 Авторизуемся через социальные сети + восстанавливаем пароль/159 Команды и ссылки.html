Установить бандл symfonycasts/reset-password-bundle:1.0:

symfony composer require symfonycasts/reset-password-bundle:1.0


Установить бандл knpuniversity/oauth2-client-bundle:2.7:

symfony composer require knpuniversity/oauth2-client-bundle:2.7


Установить бандл league/oauth2-facebook:

symfony composer require league/oauth2-facebook


Установить бандл league/oauth2-google:

symfony composer require league/oauth2-google


Создать контроллер:

symfony console make:controller


Создать пользователя:

symfony console app:add-user


Добавить поле в сущность User:

symfony console make:entity User


Настроить систему восстановления пароля:

symfony console make:reset-password


Вывести все маршруты:

symfony console debug:route


Сформировать миграции:

symfony console make:migration


Выполнить миграции:

symfony console doctrine:migrations:migrate
