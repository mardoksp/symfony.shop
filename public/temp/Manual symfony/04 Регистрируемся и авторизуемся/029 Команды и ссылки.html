Создание сущности User:

symfony console make:user



Сформировать миграции:

symfony console make:migration


Выполнить миграции:

symfony console doctrine:migrations:migrate


Создать функционал регистрации:

symfony console make:registration-form


Установить бандл symfonycasts/verify-email-bundle:

symfony composer require symfonycasts/verify-email-bundle


Создание функционала авторизации:

symfony console make:auth


Создание контроллера:

symfony console make:controller


Изменить содержимое сущности User:

symfony console make:entity User


Создание формы:

symfony console make:form


Посмотреть измененные файлы в git:

git status


Добавить все файлы в коммит:

git add .


Сделать коммит:

git commit .


Залить коммит в репозиторий на github:

git push -u origin main
