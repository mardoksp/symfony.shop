ВНИМАНИЕ:

Если вы не знаете, как работает команда, добавьте в конце команды “ --help”, например “symfony console doctrine:schema:validate --help”, чтобы посмотреть по ней документацию, не выполняя ее.


Создание базы данных (если она не была создана):

symfony console doctrine:database:create


Удаление базы данных:

symfony console doctrine:database:drop


Загрузка фикстур:

symfony console doctrine:fixtures:load


Выполнение sql-запроса:

symfony console doctrine:query:sql “...”


ВНИМАНИЕ:

Если вы добавите в конце “ --dump-sql”, то команда только выведет вам список SQL-команд, которые она собирается выполнить при использовании команды без указанного флага. Пример: “symfony console doctrine:schema:create --dump-sql” только выведет список команд.


Выполнение запросов по созданию и связке таблиц:

symfony console doctrine:schema:create


Удаление таблиц:

symfony console doctrine:schema:drop


Обновление таблиц:

symfony console doctrine:schema:update (если добавить в конце “ --force”, то команда выполнит все sql-запрос по обновлению таблиц)


Проверка валидности структур entity:

symfony console doctrine:schema:validate
