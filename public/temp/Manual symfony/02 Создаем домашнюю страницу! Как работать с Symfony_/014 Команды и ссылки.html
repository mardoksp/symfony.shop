Создать контроллер:

symfony console make:controller


Создать сущность:

symfony console make:entity


Вывести все команды MakerBundle:

symfony console list make


Установка PostgreSQL:

sudo apt update

sudo apt install postgresql postgresql-contrib


Создание базы данных и пользователя в PostgreSQL:

sudo -u postgres psql

create database ranked_choice;

create user rc_admin with encrypted password 'rc1234';

grant all privileges on database ranked_choice to rc_admin;


Создание миграции:

symfony console make:migration


Выполнение миграции:

symfony console doctrine:migrations:migrate
