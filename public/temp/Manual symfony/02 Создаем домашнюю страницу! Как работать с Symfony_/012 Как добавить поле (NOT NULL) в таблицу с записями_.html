Задача:

Добавить в сущность User поле "is_deleted", которое стандартно должно быть у всех "0"


Рекомендованный способ:

  1. Создаем новое свойство в сущности, по дефолту ставим, что быть null не может.

    symfony console make:entity User


  2. Создадим миграцию:

    symfony console make:migration


  3. Откроем миграцию и добавляем в метод up() запросы:

    ALTER TABLE users ADD is_deleted BOOLEAN NULL; (добавит новое поле с дефолтным значение "NULL")

    UPDATE users SET is_deleted='0' WHERE is_deleted IS NULL; (is_deleted='0' во всех записях, где есть это поле в значение NULL)

    ALTER TABLE users ALTER is_deleted SET NOT NULL; (проставит везде дефолтное значение NOT NULL)
